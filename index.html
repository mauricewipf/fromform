Features

Send Contact Forms to Your Email

No Server-Side Code

No need to mess with PHP or any other server side language. Just POST the form to the ACTION link mentioned above.

Server-Side validation

You're safe on backend. Feel free to add client side validation if needed.

Fully CSS Customizable

Just make sure not to change input names.
